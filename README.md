# auto-speakers

`spkreg` — маленькая утилита, которая **узнаёт голоса** на ваших записях.

## Что это и зачем

Программы для расшифровки встреч умеют делить запись на реплики и отмечать, что говорили разные люди — но называют их безлично: «Говорящий 1», «Говорящий 2». Причём на каждой новой записи нумерация своя: сегодняшний «Говорящий 1» на завтрашней встрече легко станет «Говорящим 3».

`spkreg` это чинит. Вы один раз указываете, какой голос кому принадлежит — и дальше на любых новых записях программа сама подсказывает: «этот голос на 96% похож на Ивана». Имена перестают путаться, а расшифровку можно сразу читать с настоящими именами вместо «Говорящий 1».

## Как это работает

Диаризатор (программа, которая делит запись по голосам) не только режет разговор на реплики, но и считает для каждого голоса **отпечаток** — набор из 256 чисел, описывающий тембр и манеру речи. У одного человека отпечатки на разных записях выходят похожими, у разных людей — заметно расходятся.

`spkreg` делает с этими отпечатками три простые вещи:

1. **Запоминает.** Вы один раз говорите «этот голос — Иван» — отпечаток сохраняется в список известных.
2. **Усредняет.** Если один человек встречается на нескольких записях, программа берёт все его отпечатки и считает средний. Такой «эталон» точнее и меньше зависит от случайностей одной записи — шумного микрофона, простуды, плохой связи.
3. **Сравнивает.** Для нового голоса программа смотрит, насколько его отпечаток близок к каждому эталону. Близость — это насколько два набора чисел «смотрят в одну сторону»: у одного и того же человека она около 0.99, у разных людей — около 0.44. Самый близкий эталон и есть ответ.

Внутри — только арифметика над готовыми отпечатками, никакого обучения. Весь список хранится в одном текстовом файле, его можно открыть и посмотреть глазами.

## Установка

Нужны две вещи: диаризатор, который делает отпечатки, и сам `spkreg`.

### 1. Диаризатор (whispermlx)

Работает на Mac с процессором Apple Silicon. Ставится одной командой (Python 3.12):

```bash
uv tool install --python 3.12 whispermlx \
  --with "numba>=0.61" --with "torchcodec>=0.6.0,<0.8.0"
```

Пакет и подробности: <https://github.com/KalebJS/whispermlx>

Модель, которая различает голоса, закрыта согласием — её нужно один раз открыть:

1. Заведите бесплатный аккаунт на <https://huggingface.co>
2. Откройте страницу модели <https://huggingface.co/pyannote/speaker-diarization-community-1> и примите условия использования (кнопка на странице).
3. Создайте токен доступа на <https://huggingface.co/settings/tokens> и добавьте его в окружение:
   ```bash
   export HF_TOKEN=hf_ваш_токен
   ```

После первого запуска модель скачается на компьютер, и токен больше не понадобится.

### 2. spkreg

Любой способ на выбор.

**Просто скопировать скрипт** — нужен только Python 3, никаких зависимостей:

```bash
curl -fsSL https://raw.githubusercontent.com/mainpart/auto-speakers/main/spkreg.py -o ~/.local/bin/spkreg
chmod +x ~/.local/bin/spkreg
```
(каталог `~/.local/bin` должен быть в `PATH`)

**Через pip или uv:**

```bash
pip install git+https://github.com/mainpart/auto-speakers
# или в отдельном изолированном окружении:
uv tool install git+https://github.com/mainpart/auto-speakers
```

**Через Homebrew:**

```bash
brew tap mainpart/auto-speakers https://github.com/mainpart/auto-speakers
brew trust mainpart/auto-speakers
brew install spkreg
```

## Как пользоваться

**Шаг 1. Сделать расшифровку с отпечатками.** Прогоните запись через диаризатор:

```bash
whispermlx встреча.mp3 --language ru --diarize --speaker_embeddings -f json --output_dir .
```

Получится файл `встреча.json` с репликами и отпечатками голосов. Голоса в нём пока безымянные: `SPEAKER_00`, `SPEAKER_01`…

**Шаг 2. Первый раз — представить голоса.** Послушайте запись, поймите, кто есть кто, и скажите программе:

```bash
spkreg enroll встреча.json SPEAKER_00=Иван SPEAKER_01=Мария
```

Теперь эти голоса известны. Это единственный шаг, где нужны ваши уши.

**Шаг 3. На новых записях — узнать голоса.** Для каждого голоса программа покажет самого похожего человека, уверенность в процентах и пару запасных вариантов:

```bash
spkreg identify другая-встреча.json
```

**Подставить имена прямо в текст расшифровки:**

```bash
spkreg apply другая-встреча.json -o готово.txt
```

### Все команды

| Команда | Что делает |
|---|---|
| `spkreg enroll файл.json SPEAKER_00=Имя …` | запомнить, какой голос кому принадлежит |
| `spkreg identify файл.json` | подсказать, кто говорил на новой записи |
| `spkreg apply файл.json -o out.txt` | сохранить расшифровку с настоящими именами |
| `spkreg list` | показать всех, кого программа уже знает |
| `spkreg forget Имя` | забыть голос (например, если ошиблись при разметке) |

Полезно знать: чем с большего числа **разных** встреч вы представите одного человека, тем точнее программа будет его узнавать. Двух-трёх записей обычно достаточно.

Список голосов хранится в `~/.local/share/spkreg/registry.json`.
